# Personal Web Portfolio Server

A full-stack local web application designed to host an interactive professional profile, bridging business management principles with technical implementation.

## 🚀 Project Overview
This project deploys a backend local web routing server using Python and Flask. The user interface leverages customized structural elements and inline CSS configurations to display a professional portfolio dashboard tailored for business-tech roles.

## 🛠️ Technical Stack
* **Backend:** Python 3 Core Logic
* **Framework:** Flask Web Server Engine
* **Frontend:** Structural HTML Layouts & Custom Inline CSS 
* **Environment Management:** Windows Terminal & System PATH Variables
